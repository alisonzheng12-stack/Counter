# 計算裝置

極簡日系風格的學習計算工具，可直接部署到 GitHub Pages。

## 功能

- 選擇題總答題數
- 申論題完成數
- 錯題訂正數
- 可自行設定分鐘數的倒數計時
- 倒數結束時顯示炸彈提示
- 今日目標輸入與個別達標提示
- Nippon Colors 背景、字體、框線色選擇
- 使用瀏覽器 localStorage 自動保存資料

## GitHub Pages 部署

1. 建立一個新的 GitHub repository。
2. 上傳此資料夾內所有檔案到 repository 根目錄。
3. 到 repository 的 Settings -> Pages。
4. Source 選擇 Deploy from a branch。
5. Branch 選擇 main，資料夾選擇 /root。
6. 儲存後等待 GitHub 產生網址。

入口檔案是 `index.html`。
